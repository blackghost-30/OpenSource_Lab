# 程序框架

## 程序功能

本程序实现了一个二维云台稳定器，下位机使用大疆C板(stm32f407)作为主控芯片，负责读取陀螺仪数据并控制两个电机，实现双轴增稳。

主要功能包括：
1. 读取IMU数据，获取云台的姿态信息。
   1. 使用SPI接口读取IMU原始数据(三轴角速度+三轴加速度)。
   2. 使用互补滤波算法融合加速度计和陀螺仪数据，计算出云台的姿态角(俯仰角和横滚角)。
   3. 此部分代码位于UserLib/BMI088文件夹下,IMU_Task.cpp负责统筹IMU数据读取与姿态计算，计算完成后通过RTOS任务通知通知云台控制任务(App/GimbalTask.app中)。
2. 通过PID控制算法，计算电机的控制信号，实现姿态稳定。
   1. 获取目标姿态角和当前姿态角(从IMU_Task获取，通过函数传参传入)。
   2. 使用PID控制算法计算电机输出扭矩，实现双轴稳定。陀螺仪闭环控制频率1kHz。
   3. 此部分代码封装为Gimbal类，UserLib/Gimbal文件夹下，负责核心控制算法。电机驱动代码位于UserLib/QD4310，负责使用CAN总线与电机建立通信。
3. 与上位机通信，接收控制指令。
   1. 在双轴稳定的基础上，可以使用串口通信云台的运动移动控制。串口信号由上位机发送。
   2. 云台以1kHz频率向上位机发送当前状态。
   3. 波特率：1152000（3个0），具体格式见TransmitTask.cpp文件
